
## 프로젝트 팀원 매칭 서비스 - SSAPID
--------------------------------------

#### 1. 프로젝트 소개
**1) 프로젝트 개요 :** 진행하고픈 프로젝트에 알맞는 조건을 가진 팀원들을 편하게 구할 수 있는 서비스 제공

**2) 프로젝트 기간 : 2021.01.10 ~ 2021.02.18

**3) 사용 기술 :** 
Back-End - Java, Spring Boot, Node.js
Front-End - React, Typescript, SCSS
DB : MariaDB, Firebase
Server : Ubuntu 20.04
